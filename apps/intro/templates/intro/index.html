{% trans 'Introduction' %}

{% blocktrans %} Sharif AI Challenge is a programming competition for all who are interested in artificial intelligence.

This competition is held in two phases annually. Including an online and an on-site phase in which competitors will compete in teams of three in a game designed by our technical team.

The only pre-requisite to enter this competition is familiarity with programming using C++, Java or python. But obviously, knowledge of algorithmic thinking and artificial intelligence will be a great asset for any of the participating teams. The registration for online competition is free. {% endblocktrans %}

{% trans 'FAQ' %}
{% blocktrans %} What is AI Challenge? {% endblocktrans %}

{% blocktrans %} Sharif AI Challenge is a team competition in which each team designs their strategy in order to win in a strategic challenge. These strategies are implemented as running codes. Each team attempts to devise a strong AI which can triumph against other AIs in a game in order to make it to the on-site competition. {% endblocktrans %}

{% blocktrans %} Which programming languages are supported in AI Challenge? {% endblocktrans %}
{% blocktrans %} You may write your code using Java, Python, and C++. {% endblocktrans %}
{% blocktrans %} If AI Challenge is a battle between codes, what game will the AIs be playing? {% endblocktrans %}
{% blocktrans %} The exact answer to this question will not be revealed until the beginning of the online round of the competition; however, the game this year will most likely be hero-based and of a strategic nature. {% endblocktrans %}
{% blocktrans %} Is professional knowledge of programming required in order to compete in AI Challenge? {% endblocktrans %}
{% blocktrans %} Although such knowledge would surely help, but the structure of the challenge is such that basic knowledge of programing is enough to compete. Additionally, many educational content in different levels of expertise will be made available to contestants in the course of the competition. {% endblocktrans %}
{% blocktrans %} Is AI Challenge only for Computer Science and Engineering students? {% endblocktrans %}
{% blocktrans %} Of course not. Everybody including graduated students, all the other major students and even high school student can participate in this contest. Since there is a lot of learning in this competition we recommend this contest to every student in the field of Computer Science and Engineering, Electrical Engineering, Math and Industrial Engineering who seeks to learn more and practice better. {% endblocktrans %}
{% blocktrans %} Is any prior knowledge of AI necessary? {% endblocktrans %}
{% blocktrans %} Not at all. Since many contestants have no prior experience in working with AI algorithms, the game is designed to have more than one winning strategy. The game can therefore be won using even the basic algorithms. AI-related educational content will also be made available to the contestants in the course of the competition. {% endblocktrans %}
{% blocktrans %} How many rounds is the competition composed of? {% endblocktrans %}
{% blocktrans %} The teams will be competing online from early February till early March. Friendly games will be held in this period to improve the outcomes of the teams. At the end of this stage, a selective competition will take place and the 48 top teams will make it to the on-site round of the competition. This ﬁnal round, in which the selected teams are expected to attend, will take place in mid February at Sharif University of Technology. {% endblocktrans %}
{% blocktrans %} Is attendance at Sharif University of Technology required in order to compete? {% endblocktrans %}
{% blocktrans %} The registration and the online round of the competition do not require attendance. But in order to compete in the on-site round, attendance is required. Contestants who are not from Tehran will be provided with the needed facilities. {% endblocktrans %}
{% blocktrans %} How must one register to compete in AI Challenge? {% endblocktrans %}
{% blocktrans %} You may register online starting February 5th. The registration for the online round is free. {% endblocktrans %}
